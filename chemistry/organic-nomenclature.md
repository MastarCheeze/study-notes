# Organic Nomenclature

## Prefixes

Use longest carbon chain

| # of carbon | Prefix |
| ----------- | ------ |
| 1           | meth-  |
| 2           | eth-   |
| 3           | prop-  |
| 4           | but-   |
| 5           | pent-  |
| 6           | hex-   |
| 7           | hept-  |
| 8           | oct-   |
| 9           | non-   |
| 10          | dec-   |

## Suffixes

| Family           | Suffix                                |
| ---------------- | ------------------------------------- |
| Alkanes          | -ane                                  |
| Alkenes          | -ene                                  |
| Alcohols         | -anol                                 |
| Carboxylic acids | -anoic acid                           |
| Esters           | (alcohol)-yl (carboxylic acid)-anoate |
