# Common Ions

**Cation** → (+) positively charged ion (attracted to cathode)

**Anion** → (-) negatively charged ion (attracted to anode)

## Polyatomic ions

|           |                        |
| --------- | ---------------------- |
| Oxide     | $\mathrm{O_2\\^-}$     |
| Hydroxide | $\mathrm{OH\\^-}$      |
| Carbonate | $\mathrm{CO_3\\^{2-}}$ |
| Nitrate   | $\mathrm{NO_3\\^-}$    |
| Sulfate   | $\mathrm{SO_4\\^{2-}}$ |
| Phosphate | $\mathrm{PO_4\\^{3-}}$ |

## Metal ions

|        |                    |
| ------ | ------------------ |
| Zinc   | $\mathrm{Zn^{2+}}$ |
| Lead   | $\mathrm{Pb^{2+}}$ |
| Silver | $\mathrm{Ag^+}$    |
