# Common Ions

**Cation** → (+) positively charged ion (attracted to cathode)

**Anion** → (-) negatively charged ion (attracted to anode)

## Polyatomic ions

|           |                |
| --------- | -------------- |
| Oxide     | $\ce{O_2^-}$   |
| Hydroxide | $\ce{OH^-}$    |
| Carbonate | $\ce{CO_3^2-}$ |
| Nitrate   | $\ce{NO_3^-}$  |
| Sulfate   | $\ce{SO_4^2-}$ |
| Phosphate | $\ce{PO_4^3-}$ |

## Metal ions

|        |              |
| ------ | ------------ |
| Zinc   | $\ce{Zn^2+}$ |
| Lead   | $\ce{Pb^2+}$ |
| Silver | $\ce{Ag^+}$  |
