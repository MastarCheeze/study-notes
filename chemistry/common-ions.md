# Common Ions

**Cation** → (+) positively charged ion (attracted to cathode)

**Anion** → (-) negatively charged ion (attracted to anode)

## Polyatomic ions

|           |               |
| --------- | ------------- |
| Oxide     | $\ce{O^2-}$   |
| Hydroxide | $\ce{OH^-}$   |
| Carbonate | $\ce{CO3^2-}$ |
| Nitrate   | $\ce{NO3^-}$  |
| Sulfate   | $\ce{SO4^2-}$ |
| Phosphate | $\ce{PO4^3-}$ |

## Metal ions

|           |              |
| --------- | ------------ |
| Zinc      | $\ce{Zn^2+}$ |
| Lead      | $\ce{Pb^2+}$ |
| Silver    | $\ce{Ag^+}$  |
| Aluminium | $\ce{Al^3+}$ |
